Программа для генерации excel/pdf таблицы из случайных данных взятых из api и локальных данных. Для запуска программы нужно запустить работу main класса Creator, ввести в консоль колличество людей, которых нужно занести в таблицу, после этого программа сгенерирует 2 таблицы Pdf и excel и выведет в консоль их пути.
